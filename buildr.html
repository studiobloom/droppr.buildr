Background Video(s)

Prep your video by using Adobe's Video Crop tool. You will need to create Landscape 16:9 and Portrait 9:16 versions, make sure you mute it as well! Aim for video to be under 10 seconds, Max file size is 5mb. MP4 Format is required.



Logo + Font

Upload your logo image (SVG format is best if possible). Choose a font you feel matches your logo and aesthetic from Dafont or a similar site. Download the font file and upload it here.




Nav Links

Now we'll create the top navigation menu using the font file you uploaded, your links and the names of said links. You don't want to put social media links here, they'll be placed at the bottom of the page.

Top Navigation Example:
Nav Link 1 URL: store.yoursite.droppr.xyz Text: STORE









Social Icons

The last step is adding your socials to the bottom navigation. Add the social media links you want and then head to Icons8 for free icon files (SVG is best).
